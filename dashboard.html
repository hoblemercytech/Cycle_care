Simple explanation

Last period start date: The first day blood started coming out.

Last period end date: The last day blood stopped completely.

Average cycle length: Count from the first day of one period to the first day of the next period. Many women use 28 days if they don’t know.

Average period length: How many days your period usually lasts. Many women use 5 days if they don’t know.

Don’t worry if you’re not sure. CycleCare will learn and calculate better averages as you keep logging your periods.
